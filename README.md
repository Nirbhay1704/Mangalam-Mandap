# Mangalam Mandap Services

A premium wedding decoration and event rental website.

## Features
- **Modern Homepage**: A high-end, responsive design showcase.
- **Quotation System**: An interactive tool to generate and calculate event estimates.
- **Gujarati Localization**: Specifically designed for a local audience.
